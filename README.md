# Vse Tut data mart and ad hoc task 

## Описание проекта
Проект по анализу клиентских данных компании «Все Тут».
Компания планирует внедрение ML-технологий для повышения продаж и лояльности клиентов. В рамках проекта была разработана витрина пользовательских данных и выполнен ряд ad hoc SQL-задач для исследования поведения пользователей, заказов и способов оплаты.

## Основные задачи проекта
Построить с помощью SQL витрину клиентских данных product_user_features, используя информацию о заказах пользователей. Витрина должна содержать информацию о заказах каждого клиента с учётом его региона.
Решить четыре ad hoc задачи на исследование данных в витрине и сформулировать по каждой из них выводы, которые следуют из полученных результатов.

## Пример выполненых запросов
- доля пользователей по способам оплаты
- средняя продолжительность активности пользователей
- сбор статистики с разбивкой по регионам
- ранжирование пользователей по нескольким признакам
- сегментация пользователей

## Основные выводы 
- Более половины заказов оплачиваются при помощи рассрочки
- Наибольшая доля отмен заказов наблюдается в Москве
- Наиболее высокая пользовательская активность приходится на ноябрь и декабрь
- Средняя продолжительность активности пользователей постепенно уменьшается от начала года к концу
- Средняя стоимость заказа остаётся стабильной в течение года

# Vse Tut data mart and ad hoc task 

## Описание проекта
Проект по анализу клиентских данных компании «Все Тут».

Компания планирует внедрение ML-технологий для повышения продаж и лояльности клиентов. В рамках проекта была разработана витрина пользовательских данных и выполнен ряд ad hoc SQL-задач для исследования поведения пользователей, заказов и способов оплаты.
## Используемые инструменты 
-PostgreSQL 
-SQL

## Используемые SQL конструкции 
-CTE
-JOIN
-GROUP BY
-ORDER BY 
-CASE
-агрегатные функции 
-оконные функции
-работа с датами и временем
